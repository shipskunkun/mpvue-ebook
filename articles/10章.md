##10-1 搜索页面简介

点击搜索，跳转到 search 页面
这个 path 怎么填？好像没有专门的 router 页面
需要写，page.json 中的配置信息

```javascript
this.$router.push({
  path: '/pages/search/main',
  query: {
    hotSearch: this.hotSearch
  }
})
```
